# tmux

tmux нужен прежде всего на SSH-серверах: запущенные процессы останутся жить,
если связь оборвётся. Локально Ghostty уже умеет вкладки и split panes, поэтому
tmux не обязан быть постоянной оболочкой.

```bash
tmux new -s work
tmux ls
tmux attach -t work
tmux kill-session -t work
```

Префикс настроен на `Ctrl+A`. Нажмите его, отпустите и нажмите вторую клавишу.

| Клавиши | Действие |
|---|---|
| `Ctrl+A c` | новое окно в текущем каталоге |
| `Ctrl+A \|` / `Ctrl+A -` | горизонтальный / вертикальный split |
| `Ctrl+A h/j/k/l` | перейти к соседней панели |
| `Ctrl+A H/J/K/L` | изменить размер панели |
| `Ctrl+A d` | отключиться от сессии, оставив процессы работать |
| `Ctrl+A [` | scroll/copy mode; `q` — выход |
| `Ctrl+A r` | перечитать конфиг |

`Ctrl+A Ctrl+A` передаёт literal `Ctrl+A` программе внутри tmux. Конфиг включает
мышь, историю на 100 000 строк, vi-режим копирования, truecolor и Catppuccin
статусную строку.
