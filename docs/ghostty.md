# Ghostty и prompt

Конфиг Ghostty применяется только на macOS. Он использует Catppuccin Mocha,
MesloLGS Nerd Font Mono 15pt, умеренную прозрачность/blur и block-cursor.
Для совместимости с SSH он объявляет `TERM=xterm-256color` и отключает
автоматическую shell integration: это избегает расширенных keyboard-последова-
тельностей, которые некоторые удалённые Zsh-сессии интерпретируют неверно.
После изменения конфигурации нажмите `Cmd+Shift+,` или полностью перезапустите
Ghostty.

Powerlevel10k намеренно минималистичен: одна строка, без рамок и без правого
status-блока. Локально он показывает каталог, Git и короткий prompt; ошибки
отмечаются красным prompt-символом, а не широкой правой панелью. По SSH включён
ещё более простой safe-профиль без P10K и перерисовывающих ZLE-плагинов.

```bash
ghostty +version
ghostty +list-fonts | rg -i 'Meslo.*Nerd'
```

Если icon-глифы или ширина prompt выглядят неверно, проверьте, что Ghostty
действительно использует именно Meslo Nerd Font Mono, а не fallback-шрифт.

## SSH и совместимость

По умолчанию этот репозиторий использует `TERM=xterm-256color`, который уже
есть на Ubuntu и большинстве серверов. Поэтому дополнительный terminfo на
сервере не требуется. Ранее скопированный `xterm-ghostty` можно оставить — он
безопасен, но с текущей настройкой не используется.

Если позже захочется вернуть все расширенные возможности Ghostty, сначала
проверьте конкретный сервер, а затем верните:

```bash
term = xterm-ghostty
shell-integration = detect
```
